Audio Mule 32 Channel NDI Audio Dembedder MAC Release v01
Open source built in Cycling74 Max MSP. Shoutouts to david buttler for developing the jit.ndi framework. 


Audio-Mule Standalone Executable MacOS ARM64 V1.0

Dependencies:
NDI Tools V6: https://ndi.video/tools/
libndi.dylib: https://github.com/Spacecase808/NDI-Audio-Mule/blob/main/libndi.dylib.zip

Features:
Supports Apple core audio driver
Independant source routing available
Independant VU monitoring
Universal & Independant clock source on NDI incoming streams.

How To:
1.) Launch audio-mule.app
2.) Refresh NDI sources list and select inputs
3.) Select your DSP of choice and turn on the Signal proccessing.
4.) Start streaming NDI audio to Locally connected audio DAC's including DVS / Blackhole / VSTs / UAC audio drivers

Requirements:
ARM CPU
48khz sample rate
Hardwired network interface
Physical / virtually connected DSP

.mxe included for running in Max for Live
.maxpat included for editing
